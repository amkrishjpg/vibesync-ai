# VibeSync AI

VibeSync AI is a unique music recommendation web application that suggests songs based on user mood, vibe, and listening context instead of only genre or artist selection.

## Features Planned
- Mood-based recommendations
- “Describe your vibe” input
- Context-aware playlists
- Personalized dashboard
- ML-based similarity suggestions
- Modern responsive UI

## Tech Stack
### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### ML Service
- Python
- Flask or FastAPI

### Database
- MongoDB
